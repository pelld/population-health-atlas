    What this does—and does not—show

    Practice-level prevalence can reveal conditions clustering in the same places. It cannot show whether the same individual patients have both conditions. Patient-level multimorbidity requires linked person-level records.

    Methodology

    About this analysis

    The proposed pipeline combines annual QOF prevalence, practice geography and area deprivation. For each condition, a deprivation-adjusted residual estimates whether recorded prevalence is above or below the level expected from deprivation alone.

    Condition-pair scores are correlations between adjusted practice-level rates. They identify geographic clustering, not diagnoses recorded for the same patients.

    The site uses a reproducible data pipeline so the published results can be refreshed without rebuilding the webpage.
